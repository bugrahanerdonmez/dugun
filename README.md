# Dugun - Damla & Bugrahan Wedding Website

A beautiful and personalized wedding website for Damla and Bugrahan's special day. This project showcases the couple's love story, event details, and more in an elegant, responsive design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Photo Gallery**: Display wedding photos and memories.
- **Event Details**: Include ceremony, reception, and RSVP information.
- **Guest Book**: Allow guests to leave messages.
- **Customizable**: Easy to modify colors, fonts, and content.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dugun.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dugun
   ```
3. Open `index.html` in your web browser or serve it using a local server (e.g., with Python: `python -m http.server`).

## Usage

- Edit the HTML, CSS, and JavaScript files to customize the content.
- Add your own images to the `gallery/` folder.
- Deploy to a web hosting service like GitHub Pages, Netlify, or Vercel.

- HTML5
